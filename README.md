# [Acessar Página](https://react-album-nu.vercel.app)

## 🖥️ Projeto

Projeto album, consumindo api jsonplaceholder.

## 🧱 Tecnologias

+ Reactjs
+ Typescript
+ Tailwind
+ Axios
+ React-Router


## 🏠 Executar localmente

Clone o projeto

```bash
git clone https://github.com/FabioPenedo/react-album
```

Entre na pasta do projeto

```bash
cd react-album
```


Instale as dependências

```bash
npm install
```

Inicie o servidor

```bash
npm run dev
```
